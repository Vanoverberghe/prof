Comment écrire une page Web?

Le langage utilisé est HTML5 (HyperTexte Markup Language) pour le fond et CSS3 pour la forme.

Principe du langage HTML

Il permet de structurer la page, d'y mettre du contenu en utilisant des balises.
Le site W3Schools presente toutes les balises ainsi que les autres langages utilisés pour développer des pages web.
